App Description:
This app helps users keep track of the calories of meals they consumed for a given day. Users can create an account, log eaten meals, and view a summery of calorie information for eaten meals. 

API Links:
We used the Nutritionix API (https://developer.nutritionix.com/docs/v2), an API used to process natural language and return nutrient information from inputted text. 

API Endpoint Used: https://trackapi.nutritionix.com/v2/natural/nutrients
